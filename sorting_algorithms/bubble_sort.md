### What is a Bubble Sort? 

This is a type of algorithm that takes an unsorted list and sorts in an ascending order. This is done by comparing two values at any given time. 

For example if we have a list such as `[5, 20, 10, 25, 1, 7]`, on the first iteration, we compare `5 and 20`. It stays the same. Then onto `20 and 10`, this gets swapped. Next is `20 and 25`, then `25 and 1` which gets swapped. Lastly, we have `25 and 7` which gets swapped. 

This gives us a list `[5, 10, 20, 1, 7, 25]`. This is still not sorted, and so this process gets repeated over and over until it is sorted.
